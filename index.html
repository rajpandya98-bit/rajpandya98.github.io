Featured Project: Fall Detection System

Figure: Circuit schematic showing ADXL335 accelerometer interfacing with LM358-based signal conditioning and microcontroller input.

Designed a low-cost fall detection system using a 3-axis accelerometer (ADXL335) and analog signal conditioning (LM358).

  • Implemented filtering and amplification for stable ADC input
  • Developed threshold-based fall detection logic
  • Tested system performance across daily activities and simulated falls

The schematic was designed to ensure reliable analog signal acquisition by incorporating appropriate filtering and amplification stages prior to analog-to-digital conversion. Design decisions focused on minimizing noise and improving signal stability during dynamic motion conditions.

Testing & Validation

  • Bench-tested acceleration response across X, Y, Z axes
  • Verified threshold response for simulated fall events
  • Evaluated false positives during normal activities (walking, sitting)

Limitations observed include sensitivity to abrupt non-fall movements, highlighting the need for more advanced classification methods in future iterations.
